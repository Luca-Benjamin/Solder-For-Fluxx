# Fluxx AI - Chrome Extension

AI-powered form editing for Fluxx Grant Management System.

## Installation

1. **Download**: Click "Code" → "Download ZIP" and extract it
2. **Open Chrome Extensions**: Go to `chrome://extensions`
3. **Enable Developer Mode**: Toggle in the top right corner
4. **Load Extension**: Click "Load unpacked" and select the extracted folder
5. **Done!** The Fluxx AI icon will appear in your extensions bar

## Usage

1. Navigate to your Fluxx instance
2. Go to **Admin Panel → Form Builder** and select a form/theme
3. Click the Fluxx AI extension icon to open the side panel
4. Describe the changes you want in natural language

### Example Commands

- "Add a new section called Contact Info after Section 2"
- "Rename the Organization Details group to Company Profile"
- "Make Q1.7 only show if Q1.4 is Yes"
- "Add fields for name, email, and phone to the Contact section"

### Document Upload

You can also upload a PDF or Word document using the upload button - the AI will automatically generate a form structure based on the document content.

## Requirements

- Google Chrome browser
- Access to a Fluxx instance with Form Builder permissions

## Troubleshooting

**"Not connected" in side panel**
- Make sure you're on a Fluxx form editor page (Admin → Form Builder)
- Try refreshing the page

**Changes not applying**
- Check that you're in preview mode, not builder mode
